# TDD Kit - Ruby

## Initialization

    rvm install 2.2.1
    bundle install

## Run unit tests

    rake test

## Run lint, unit tests and coverage

    rake

## Libraries documentation

* Assertion api: [Minitest expectations]
* Linter: [Rubocop]
* Cover: [SimpleCov]
* Mock api: [Mocha]

[Rubocop]: https://github.com/bbatsov/rubocop
[SimpleCov]: https://github.com/colszowka/simplecov
