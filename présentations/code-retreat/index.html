Code Retreat

Michael Borde / @michael_borde / arpinum.fr

L'objectif ?

Réduire l'écart entre comment je code et comment j'aimerais coder.

Le déroulement

  • Session de code de 45 min
  • Suppression du code
  • Débriefing
  • On recommence

Conception simple

  • Les tests passent
  • Le code exprime l'intention
  • Code concis
  • Aucune duplication

Jeu de la vie

  • Automate cellulaire
  • John Horton Conway (1970)
  • Règles simples
  • Motifs complexes

Le métier

  • Une cellule est vivante ou morte
  • Une cellule a 8 voisines
  • La grille est infinie
  • Une graine
  • Des générations

Les règles

  • Cellule vivante avec < 2 voisines meurt
  • Cellule vivante avec > 3 voisines meurt
  • Cellule vivante avec 2 ou 3 voisines reste en vie
  • Cellule morte avec exactement 3 voisines devient vivante
